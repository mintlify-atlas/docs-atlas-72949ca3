This page answers the most common questions from new and returning players. If you have a question that is not covered here, you can ask in the [fheroes2 Discord server](https://discord.gg/xF85vbZ) or open an issue on the [GitHub repository](https://github.com/ihhub/fheroes2).

<AccordionGroup>
  <Accordion title="What is fheroes2?">
    fheroes2 is a recreation of the Heroes of Might and Magic II game engine. It is a free, open source, multi-platform project written entirely from scratch. Its goal is to reproduce the original game with significant improvements in gameplay, graphics, and logic — including support for high-resolution graphics, improved AI, and numerous UI improvements — breathing new life into one of the most beloved turn-based strategy games.
  </Accordion>

  <Accordion title="Is fheroes2 a mod?">
    No. fheroes2 is **not** a mod. Mods are reverse-engineered modifications of existing game files. fheroes2 is written from scratch and does not modify any files from the original game. It is a standalone reimplementation of the game engine that reads the original game's data files in order to provide content.
  </Accordion>

  <Accordion title="Is fheroes2 free?">
    Yes, fheroes2 itself is completely free and open source. However, it requires the data files (graphics, maps, sounds) from the original Heroes of Might and Magic II game, which you must obtain separately. You can purchase the original game from GOG or Ubisoft, or use the free demo version available from the Internet Archive.
  </Accordion>

  <Accordion title="Are my save files from the original Heroes of Might and Magic II compatible with fheroes2?">
    No. fheroes2 does not support save files from the original Heroes of Might and Magic II. The two engines use different save formats, and there is no conversion path between them. You will need to start new games in fheroes2.
  </Accordion>

  <Accordion title="How can I contribute to the project?">
    fheroes2 is a collaboration of developers, testers, artists, translators, and players. There are many ways to get involved:

    - **Development** — Fix bugs or implement features. Basic C++ knowledge is helpful but not always required. Issues labeled **Good first issue** on GitHub are a good starting point. All pull requests go through code review and automated CI/CD checks.
    - **Testing** — Play the game, report bugs, and test pull requests before they are merged. Testing requires patience and attention to detail rather than programming skills.
    - **Design and art** — The team is working on a new expansion and needs pixel artists to create new content. Contact the team on Discord for details.
    - **Sound and music** — The original game is missing sounds for some map objects, and the upcoming expansion needs new audio content. Ask on Discord for details.
    - **Translation** — English, Polish, Russian, French, and German are currently supported, and additional languages are planned. Existing translations still have gaps that need to be filled.
    - **Story and narrative writing** — The upcoming expansion campaign needs detailed story content and side quests. Contact the team on Discord for details.
    - **Voice acting** — The expansion campaign will require voice actors. Ask on Discord for details.
    - **Video** — The expansion may require cinematic video content. Ask on Discord for details.
    - **Support** — Spread the word about the project or support it financially via [Patreon](https://www.patreon.com/fheroes2).
  </Accordion>
</AccordionGroup>
