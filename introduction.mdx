fheroes2 is an open source recreation of the **Heroes of Might and Magic II** game engine, written entirely from scratch. The project is designed to reproduce the original game faithfully while introducing significant improvements in gameplay, graphics, and logic — including high-resolution graphics, smarter AI, and dozens of quality-of-life fixes. It was initially developed on SourceForge and is now actively maintained as a community effort on GitHub.

<Note>
  fheroes2 is a game **engine** recreation. It does not include the original game's assets. To play, you must provide a legally obtained copy of Heroes of Might and Magic II (full or demo version). See [Requirements](/getting-started/requirements) for details.
</Note>

## What fheroes2 is

fheroes2 reimplements the entire HoMM2 engine from scratch. When you run fheroes2, you get the original game experience plus improvements that were not possible in the original — such as playing at modern screen resolutions, an overhauled AI, a built-in map editor with a random map generator, and localization into over 20 languages.

The project is licensed under the **GNU General Public License v2.0** and is free to download and use.

## What fheroes2 is not

fheroes2 is **not a mod**. Mods are reverse-engineered modifications of existing game files. fheroes2 does not modify any files from the original game — it is an independent engine that reads the original game's resource files to provide the game content.

fheroes2 also does not include any copyrighted assets from the original game. All rights to the original game and its resources belong to Ubisoft (formerly The 3DO Company).

## Supported platforms

fheroes2 runs on:

- Windows (x64 and x86)
- macOS
- Linux (including Flatpak, Debian/Ubuntu, Arch, Gentoo, and more)
- Android
- iOS
- PlayStation Vita (requires custom firmware)
- Nintendo Switch (requires custom firmware)
