fheroes2 is an open source recreation of the **Heroes of Might and Magic II** game engine, written entirely from scratch. It breathes new life into one of the most beloved turn-based strategy games — with high-resolution graphics, smarter AI, dozens of quality-of-life improvements, and support for Windows, macOS, Linux, Android, iOS, PlayStation Vita, and Nintendo Switch.

## Get up and running

<Steps>
  <Step title="Get the original game assets">
    You need resources from **Heroes of Might and Magic II** to play. Purchase the game on [GOG](https://www.gog.com) or [Ubisoft Store](https://store.ubi.com), or download the free demo using the bundled scripts.
  </Step>
  <Step title="Download and install fheroes2">
    Choose your platform and install the latest release. See the [installation guide](/getting-started/install-windows) for step-by-step instructions for Windows, macOS, Linux, Android, and handheld devices.
  </Step>
  <Step title="Extract game resources">
    Run the included `extract_homm2_resources` script (or use the installer option) to copy game data from your HoMM2 installation into fheroes2.
  </Step>
  <Step title="Play">
    Launch fheroes2 and enjoy the classic game with modern improvements — higher resolutions, improved AI, better UI, and much more.
  </Step>
</Steps>
