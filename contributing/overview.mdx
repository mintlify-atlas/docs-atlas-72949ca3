fheroes2 is a community-driven project built by developers, artists, translators, testers, and players from around the world. Every contribution matters — whether you're fixing a bug, translating a string, drawing a pixel, or simply spreading the word. This page explains every area where you can make a difference.

<Note>
  The best place to coordinate with the team and ask questions is the [fheroes2 Discord server](https://discord.gg/xF85vbZ). Many contribution areas (art, sound, story, voice acting) are actively discussed there.
</Note>

## Development

Development is the largest contribution area. You do not need to be an expert C++ programmer to get started — many open issues require only basic programming skills.

- **Good first issues** — tasks tagged [`good first issue`](https://github.com/ihhub/fheroes2/issues?q=is%3Aissue%20is%3Aopen%20label%3A%22good%20first%20issue%22) are scoped specifically for newcomers.
- **Help wanted** — tasks tagged [`help wanted`](https://github.com/ihhub/fheroes2/issues?q=is%3Aissue%20is%3Aopen%20label%3A%22help%20wanted%22) are open to any contributor.
- **CI/CD pipeline** — every pull request runs through an automated pipeline that checks for common issues before review.
- **Code review** — all pull requests go through a code review process to maintain source quality.

To build fheroes2 from source, see the [building guide](/contributing/building) or the [CMake guide](/contributing/cmake).

## Testing

The project ships frequent changes, and there is always more to test than the core team has time for. Testing contributions include:

- Discovering unreported bugs and missing features.
- Reproducing and confirming existing issues.
- Testing open pull requests before they are merged.

Testing requires patience and attention to detail rather than programming knowledge, making it a great entry point for non-developers.

## Design and art

The fheroes2 team is working on a brand new add-on that requires original pixel art. Artists help with:

- Designing new in-game sprites and assets, starting with simpler single-image tasks.
- Contributing to the `resurrection.h2d` data file used for new graphical content.

For a full guide to the asset pipeline, see the [graphical assets guide](/contributing/graphical-assets).

<Note>
  Coordinate with the art team in the [Discord server](https://discord.gg/xF85vbZ) before starting new assets to avoid duplication of effort.
</Note>

## Sound and music

The original game is missing sounds for certain map objects. The upcoming add-on also requires entirely new sound effects and music tracks. If you are a composer, sound designer, or audio producer, reach out on [Discord](https://discord.gg/xF85vbZ) to discuss what is needed.

## Translation

fheroes2 supports Polish, Russian, French, German, and many more languages through PO translation files. Existing languages still have incomplete translations, and new languages are added regularly.

For full instructions on how to contribute translations, see the [translation guide](/contributing/translation).

## Story writing

The campaign in the upcoming add-on needs a detailed narrative — including side quests and dialogue — built on top of the team's initial story sketches. If you enjoy narrative writing, reach out on [Discord](https://discord.gg/xF85vbZ) to learn more about the campaign direction.

## Voice acting

The add-on campaign will feature voice acting to bring the story to life. Voice actors of any background are welcome to audition. Contact the team on [Discord](https://discord.gg/xF85vbZ) for details on the roles available.

## Support

Not a programmer, artist, or translator? You can still make a real difference:

- **Spread the word** — share fheroes2 with friends, post about it, or write a review.
- **Donate** — contributions through [Patreon](https://www.patreon.com/fheroes2) go directly toward the project's future improvements.
